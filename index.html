A little bit about me.

I call Minneapolis, Minnesota home, but I'm always looking for my next travel destination. My journey into tech started as a healthcare provider moving my practice into virtual space. As I started working exclusively online, I was fascinated with how easily I could connect with people. I was hooked and wanted to learn how to create a site to fit my own specific needs.

That curiosity has only amplified with time. When I learned how to build my first website, I realized I found something that gave me the freedom & versatility I was looking for in my work. Now I've made a full switch to front-end development, where I can use my exceptional organization skills and eye for detail to write clean, elegant code.

Hand-Coded HTML

My focus is writing clean, well-formatted, semantic HTML5 by hand to make sure that the content is easy to read, easy to collaborate, trouble-shoot and accessible.

Well-Organized CSS

I pride myself on writing CSS that is easy to read and build on. I focus on keeping my CSS lean and fast to load, and I make it a habit to stay up to date on current best practices.
